Food webs

  • Predator-prey body mass ratio for machine-learned groups.
    1. Identify \(k_i\) groups based on k-mean clustering on traits or other table.
    2. Fit the food web inference model with each group having a predator-prey optimal scaling ratio.
    3. Repeat using \(k_{i+1} = k_i + 1\)
    4. Repeat for several \(k\).
    5. Compare each inference model using AUC or Boyce index.
    6. Select the best model.
    7. Explore how the species were partitioned into groups.
